# Mini Chess Clock

A lightweight dual timer for two players, inspired by chess clocks.  
Each player starts with **5 minutes**. The timer switches after each move, demonstrating countdown timers and state management.

## Features
- Two 5‑minute countdown timers.
- Switches active timer when a player clicks **End Turn**.
- Alerts when a player’s time runs out.
- Reset button to restart the game.
- Clean dark theme with gradient buttons.
