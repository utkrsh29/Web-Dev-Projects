# Career Roadmap Planner

An interactive learning path visualizer helping beginners and students chart progress across multiple fields (Frontend Developer, Backend Developer, Data Scientist, AI/ML Engineer).

## Features

- **Selectable Career Paths**: Pre-defined learning trees for major tech fields.
- **Interactive Timelines**: Skill nodes mapped sequentially through progressive learning phases (e.g. Beginners, Intermediate, Advanced).
- **Interactive Checklists**: Mark skills completed to dynamically update phase progress blocks and overall roadmap completion meters.
- **Interactive Side Information Hub**: Hovering or clicking any node displays a detailed description, learning checkpoints, and curated resource links.
- **State Persistence**: Selected roadmaps and completed nodes save in `localStorage`.
- **Fully Responsive Grid**: Timeline components adapt to layout constraints on mobile screens, collapsing into a linear roadmap.

## Technologies Used

- **HTML5**: Structured dashboards.
- **CSS3**: Variables, flexbox/grid containers, and animations.
- **JavaScript (ES6+)**: OOP Career model loaders and event listeners.

## Installation

Clone the repository and open `index.html` inside a web browser:
```bash
git clone https://github.com/MistryVishwa/Web-Dev-Projects.git
cd "Projects/Career Roadmap Planner"
open index.html
```

Or serve the directory using a local HTTP server:
```bash
npx http-server
```
