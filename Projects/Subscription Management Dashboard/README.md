# Subscription Management Dashboard

A premium dark finance dashboard for tracking and managing subscriptions. Built with vanilla HTML, CSS, and JavaScript.

## Features
- **Real-time metrics**: Monthly spend, annual spend, and active subscription count update instantly
- **Category distribution bars**: Visual colored meter blocks showing spending ratios
- **Renewal alerts**: Cards glow with coral warning when renewal is within 7 days
- **Filter & sort**: Filter by category, sort by name/cost/renewal date
- **Persistent storage**: All data saved to localStorage
- **Dark finance theme**: Deep slate backdrop with neon violet accents
