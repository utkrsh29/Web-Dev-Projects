# Word Scramble Game

A timed arcade word puzzle built with plain HTML, CSS and JS.

## Features

- 20 words with dictionary-style hints.
- Fisher-Yates shuffling algorithm with safety checks (scrambled never matches original).
- 30-second countdown timer with color-coded progress bar.
- Score system with time bonus and hint penalty.
- Persistent high-score via localStorage.
- Responsive design for mobile and desktop.

## Run it

Open `index.html` in any modern browser.
