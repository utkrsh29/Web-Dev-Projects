# Brick Breaker Arcade Game

A cyberpunk-themed arcade breakout game built with HTML5 Canvas and vanilla JavaScript.

## Features

- Hardware-accelerated 60fps render loop via `requestAnimationFrame`.
- Multi-tier brick matrix: top 3 rows (2-hit neon pink), bottom 3 rows (1-hit neon cyan).
- Dynamic ball deflection based on paddle impact position.
- Score and persistent high-score via `localStorage`.
- Mouse, keyboard (arrow keys / A, D), and touch input support.
- Victory / Game Over overlay with instant board reset.

## Run it

Open `index.html` in any modern browser.
